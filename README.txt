Instalación de la api:
1.Vamos a "admira-showroomctrl\complete\out\artifacts\complete_jar"
2.Encontraremos un archivo llamado complete.jar, creamos un acesso directo.
3.Movemos elacceso directo a la carpeta Startup de Windows.
4.Comprobamos que el PC tenga instalado Java, en caso que no lo tenga, se devera instalar JAVA SDK 8.
5.Desactivamos el Firewall de Windows.
6.Ejecutamos el archivo para comprobar que todo funcione y estara listo.
